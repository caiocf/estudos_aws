
Para gera os dados:

No Linux/macOS:
```shell
python3 -m venv venv
source venv/bin/activate
```


No Windows:
```shell
python3 -m venv venv
venv\Scripts\activate
```

Instalando dependencias:
```shell
python.exe -m pip install --upgrade pip
pip install --upgrade pip
pip install -r requirements.txt
```

Gerando o arquivo:
```shell
python gera_dados_parquet.py
```
